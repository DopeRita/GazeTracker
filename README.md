# Gaze Tracker – RGB Camera Eye Tracking

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Real-time eye tracking using only a built-in RGB camera (no dedicated hardware). Supports natural head movements (pitch/yaw/roll) and improves gaze accuracy in natural scenes through multi-pose calibration and head movement compensation.

## Key Features

- Head-free tracking (no head fixation required)
- Multi-pose calibration (front, slight yaw, slight pitch)
- Real-time performance
- Robust to pupil distortion and eyelid occlusion

## Tech Stack

- Python 3.8+
- OpenCV
- MediaPipe
- NumPy / SciPy
